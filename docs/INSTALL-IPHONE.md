# Install On iPhone Safari

1. Install the app `Userscripts` from the App Store.
2. In the Files app, create or choose the Userscripts script folder.
3. Put `userscripts/vida-helper.user.js` inside that folder.
4. Open iPhone Settings.
5. Go to Safari > Extensions > Userscripts.
6. Enable it and allow it on `vida.hmg.com`.
7. Open `https://vida.hmg.com/` in Safari.
8. After login, tap the small `VIDA` button at bottom right.

If the button does not appear, check that the Userscripts extension has website
permission for `vida.hmg.com`.
